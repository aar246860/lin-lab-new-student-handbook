## 研究室手冊

這個網站現在分成兩層。第一層是完整的「研究室手冊」，說明日常運作、研究訓練、論文發表與 AI Agent 工作流。第二層是 8 週入門訓練，讓新生用具體任務走過資料、模型、圖表與決策可靠度。

## 核心流程

```text
現地資料或教學資料
→ 整理成乾淨表格
→ 畫出洩降圖
→ 套用傳統解析模型
→ 檢查殘差與模型假設
→ 得到模型條件下的參數
→ 比較轉換不確定性
→ 推到工程決策指標
→ 完成可靠度稽核報告
```

## 學生需要知道的第一件事

研究通常不是一開始就想出很大的題目。更多時候，研究從一個很實際的問題開始：這筆資料到底能不能支持這個結論？這套教材會把這個問題拆成八週可以練習的動作。

## 資料與模型邊界

這裡使用的入門資料是合成資料，只用於教學與研究訓練。教材中的簡化公式不能直接拿來做正式現地分析，也不能被描述為完整 LDL 模型的驗證結果。
