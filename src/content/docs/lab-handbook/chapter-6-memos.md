# 第六章 備忘

本章收錄研究室日常雜項。原則很簡單：公用物品要維護，使用紀錄要清楚，遇到故障不要硬拆，重要資料不要只留在個人電腦。

## 6.1 列印與掃描

使用印表機、掃描器或影印設備時，請注意：

1. 大量列印前先確認格式、頁碼、雙面列印與紙張大小。
2. 印論文或正式文件前，先印少量檢查版面。
3. 卡紙時先停止列印，不要用力拉扯。
4. 不確定如何排除故障時，先詢問，不要自行拆機。
5. 掃描重要文件時，確認解析度、頁面順序與檔名。
6. 掃描後若需要文字搜尋，可再做文字辨識。

## 6.2 紙張與耗材

1. 可再利用的紙張放在固定位置，不和正式用紙混在一起。
2. 已含個資、成績、審稿意見或未公開研究內容的紙，不可當回收紙重複使用。
3. 碳粉、墨水、釘書針、資料夾等耗材快用完時，提早告知。
4. 公用耗材不得帶離研究室作私人用途。

## 6.3 公用設備

常見公用設備包括螢幕、轉接頭、延長線、簡報器、掃描器、外接硬碟、書籍與工具。借用時請做到：

1. 借用前確認是否有人正在使用。
2. 借走時留下紀錄。
3. 使用後放回原位。
4. 發現故障或缺件，立即記錄並告知。
5. 不把公用設備長期放在個人座位。

## 6.4 文件裝訂與歸檔

正式文件、論文、計畫報告或重要講義需要保存時：

1. 檢查封面、目錄、頁碼、圖表、參考文獻與附錄。
2. PDF 與 Word 檔都要保存。
3. 若有紙本，放到指定位置並標示題名、年份與作者。
4. 若是計畫或投稿相關文件，須保留 clean version、marked version 與回覆紀錄。

## 6.5 離開研究室前檢查

每天離開前簡單檢查：

- 電腦與螢幕是否關閉或進入安全狀態。
- 重要資料是否已儲存與備份。
- 公用設備是否歸位。
- 桌面是否留下含私人資訊或未公開研究內容的紙張。
- 門窗、燈、冷氣與電源是否依研究室規定處理。

## 6.6 畢業或離職交接

離開研究室前，至少交接：

1. 研究資料與原始檔。
2. 程式與執行說明。
3. 圖表與製圖資料。
4. 文獻庫與閱讀筆記。
5. 投稿、審稿或計畫文件。
6. 未完成工作與下一步建議。
7. 借用物品歸還紀錄。

交接的目的不是形式，而是讓下一個人能接得下去。
