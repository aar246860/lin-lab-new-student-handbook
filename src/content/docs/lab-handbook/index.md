# 研究室手冊總覽

這個版本先把舊版地下水研究室手冊建立成 private 完整 OCR 稿，再由去識別完整稿切分成公開章節。它不是摘要版，也不是重新發明一套漂亮但失真的教材；目前的核心工作是保留原手冊的順序、規則、語氣與頁碼，讓後續可以逐頁校對、逐段最小修改。

學生閱讀時，請先看章節版；需要追溯原文時，再看「完整去識別母稿」。整理版頁面只是後續訓練材料，不應被視為原手冊逐字版本。

## 目前版本的限制

1. 章節版來自 OCR 工作稿，已去識別，但尚未逐頁人工校對。
2. 章節內的錯字、簡繁混雜、標點與表格錯位，先保留為來源狀態，不在這一輪擅自改寫。
3. 含聯絡資訊、舊內部路徑、通訊錄或不宜公開內容的地方，以 `[REDACTED_*]` 標記替代。
4. 第 3 章含外部寫作書籍或教材摘要，公開使用時須再檢查引用與公開邊界。
5. 「整理版與訓練材料」是後續適合林研究室使用的教學版本，不等於舊手冊原文。

## 後續編修原則

每一次調整都應先回答三個問題：

1. 這段內容來自哪一個 OCR source page？
2. 這段內容是否已去識別且適合公開？
3. 這次修改是校正 OCR、刪除敏感資訊、最小限度現代化，還是新增林研究室內容？

只有能回答這三個問題，才把文字放進正式手冊。這樣可以避免手冊再次變成過度精簡、語氣像 AI 摘要、且無法追溯來源的版本。
