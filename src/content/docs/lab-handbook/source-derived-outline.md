# 來源對照表

這一頁不是公開 OCR 原文，而是把舊手冊的章節功能轉成可檢查的對照表。公開版手冊應該保留舊手冊的詳細訓練精神，但刪除私人資訊、過時路徑、舊制度與不適合直接公開的人名或工具清單。

## 對照原則

| 原則 | 做法 |
|---|---|
| 保留功能 | 保留「新人該做什麼、何時交、交什麼、如何命名、如何被檢查」這類訓練功能。 |
| 改寫語境 | 把舊式紙本、單機軟體與研究室內部規定，改成 GitHub、雲端資料夾、AI Agent、Python、Zotero 與現代論文工作流。 |
| 不公開原文 | OCR 僅作本機判讀，不把逐字內容、私人路徑、舊通訊資訊或過時內部細節放上網站。 |
| 標明新增 | 與林研究室現行研究主線有關的 Lagging Darcy、轉移不確定性、可靠度與 AI 工作流，屬於新版擴充，不是舊手冊原文。 |

## 使用方式

學生閱讀公開手冊時，先看「林版改寫方向」；助理或 agent 要擴寫頁面時，先回到此表確認對應的舊手冊功能。若新增內容不是由舊手冊轉譯，而是林研究室的新研究主線，必須清楚標為新版擴充。
