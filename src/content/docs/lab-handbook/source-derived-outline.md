# 來源對照表

這一頁不是公開 OCR 原文，而是說明公開手冊如何對應舊手冊。新版不再以「重新設計」為主，而是以原章節、原條列與原手冊語氣為主。

## 對照原則

| 原則 | 做法 |
|---|---|
| 保留章節 | 盡量保留舊手冊的章節編號與標題，例如 1.1、1.2、1.5、2.3、2.5。 |
| 保留條列 | 保留「須知」「注意事項」「要點」式條列，不改成短版網站摘要。 |
| 最小修改 | 只修 OCR 錯字、私人資訊、舊路徑與必要現代工具註記。 |
| 保留舊工具脈絡 | 舊手冊列出的 Fortran、MATLAB、Grapher、Surfer、Tecplot、EndNote 等工具，原則上保留其訓練脈絡。 |
| 標出新增 | Lagging Darcy、轉移不確定性、可靠度與 AI Agent 屬於林研究室新增項目，須另外標明。 |

## 使用方式

學生閱讀公開手冊時，先依章節照做；助理或 agent 要擴寫頁面時，先回到此表確認對應 OCR 頁與原章節。若某段不是舊手冊內容，而是林研究室新增訓練，須明確標示為新增項目。
