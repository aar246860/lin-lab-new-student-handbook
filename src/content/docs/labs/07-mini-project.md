# Week 7 小型專題組裝

## 學習目標

把前面幾週的練習整理成一個研究故事。報告不需要很長，但每個段落都要回答一件清楚的事：資料是什麼、模型是什麼、結果是什麼、限制是什麼、對決策有什麼影響。

## 本週任務

1. 選定一個主問題。
2. 整理資料、模型、參數與決策結果。
3. 完成一張主圖。
4. 寫出 5 頁以內報告初稿。

## 繳交成果

- `draft_report.md`：報告初稿。
- `main_figure.png`：主圖。
- `reproducibility_checklist.md`：資料與程式清單。

## 通過標準

- 報告有摘要、方法、結果、討論與限制。
- 主圖能單獨說明核心發現。
- 每個重要結論都有圖、表或計算支持。
- 有清楚標示合成資料與教學簡化。

## 給學生的提醒

好的報告不是把所有東西都塞進去，而是讓讀者知道你回答了哪個問題，也知道哪些問題還不能回答。
