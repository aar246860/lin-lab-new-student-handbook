# Week 1 資料整理與洩降圖

## 學習目標

先學會把資料整理到別人看得懂、也查得到來源。研究的第一步不是急著套模型，而是確認欄位、單位、缺值、時間順序和圖表是否清楚。

## 本週任務

1. 讀取 `data/synthetic_drawdown.csv`。
2. 說明每個欄位的單位與意義。
3. 畫出時間與洩降量的關係圖。
4. 判斷這筆資料能不能直接拿來做傳統抽水試驗分析。

## 繳交成果

- `notes.md`：資料欄位說明。
- `figure.png`：洩降時間序列圖。
- `answer.md`：100 到 200 字說明這筆資料的限制。

## 通過標準

- 圖上要有座標軸名稱與單位。
- 資料來源標示為合成教學資料（synthetic teaching dataset）。
- 文字沒有把合成資料說成現地觀測。
- 能指出至少一個資料品質檢查項目。

## 給學生的提醒

如果 Python 還不熟，先用 Excel 畫圖也可以。這週先把資料和圖整理清楚，程式技巧可以慢慢補。
