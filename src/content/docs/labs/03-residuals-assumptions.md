# Week 3 模型假設與殘差

## 學習目標

學會分辨「曲線看起來很貼」和「模型假設足以支持判斷」的差異。殘差不是附屬品，而是用來檢查模型是否漏掉重要機制的線索。

## 本週任務

1. 使用 Week 2 的傳統模型預測洩降。
2. 計算觀測值減預測值的殘差。
3. 畫出殘差對時間的圖。
4. 判斷殘差是否在早期或晚期呈現系統性變化。

## 繳交成果

- `residual_figure.png`：殘差圖。
- `answer.md`：指出殘差是否隨時間呈現系統性變化。
- `notes.md`：列出可能造成殘差結構的水文或試驗因素。

## 通過標準

- 有清楚定義殘差的計算方向。
- 至少提出兩個可能原因，例如井儲、邊界、洩漏、非均質或延遲響應。
- 沒有把殘差直接當成某一個機制的證明。
- 能說明下一步需要比較模型或補充資料。

## 給學生的提醒

看到殘差有形狀，不代表你已經知道原因。比較好的寫法是先說「這表示模型沒有解釋掉某些結構」，再小心提出可能機制。
