# Week 5 轉換不確定性

## 學習目標

理解抽水試驗不是直接量到 `T`、`S` 或 `K`。我們是先量到洩降反應，再透過模型把它轉成參數；模型不同，估出來的參數也可能不同。

## 本週任務

1. 選擇至少兩個模型或兩種判釋方式。
2. 對同一筆資料估算參數。
3. 建立一張比較表。
4. 討論差異是否會影響後續工程判斷。

## 繳交成果

- `parameter_table.csv`：模型、假設、參數、單位、備註。
- `figure.png`：不同模型對資料的擬合或解釋比較。
- `answer.md`：說明「轉換不確定性」的意思。

## 通過標準

- 能區分「量到的反應」與「估出來的參數」。
- 至少列出兩種模型假設差異。
- 不把其中一組參數稱為絕對正確答案。
- 能指出參數差異可能影響哪些工程決策指標。

## 給學生的提醒

這週不是要證明哪個模型比較厲害，而是練習一個習慣：每次把資料轉成參數，都要問「這個轉換過程帶進了哪些不確定性？」
