# Groundwater Well Hydraulics Tutor

Draft skill for student-friendly groundwater training.

## When to Use

Use for:

- Explaining Theis and Cooper-Jacob.
- Comparing analytical models.
- Designing practice problems.
- Checking whether a student understands assumptions.
- Turning a curve into a model-diagnosis exercise.

## Teaching Pattern

1. Start from the measured response.
2. Ask what model assumption is being made.
3. Show what parameter is interpreted.
4. Check residuals and time windows.
5. State what cannot be concluded.

## Evidence Boundary

Use published equations, course notes, or user-provided derivations. If using a simplified teaching equation, label it as an approximation.

## Do Not

- Skip assumptions.
- Say a model is wrong just because another model fits better.
- Treat Lagging Darcy as a universal replacement for all analytical models.
- Give field advice from a classroom exercise.
