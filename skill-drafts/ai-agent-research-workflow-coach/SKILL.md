# AI Agent Research Workflow Coach

Draft skill for structuring AI-assisted research loops.

## When to Use

Use when a task needs repeated AI support, such as:

- Literature scouting.
- Data cleaning.
- Figure regeneration.
- Manuscript revision.
- Review response.
- Website content updates.

## Standard Loop

1. Define the research goal.
2. Define source files and forbidden claims.
3. Split roles: scout, builder, critic, polisher, archivist.
4. Require a checked output.
5. Store prompt, output, accepted changes, and evidence notes.

## Evidence Boundary

Agent outputs are drafts. The final scientific judgment remains with the researcher.

## Do Not

- Let one agent both generate and approve a scientific claim.
- Skip source tracking.
- Publish generated text without checking privacy, claims, and provenance.
- Use generic AI language when domain-specific groundwater wording is required.
